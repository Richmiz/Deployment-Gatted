# Results Directory

This directory contains lightweight outputs from the completed Deployable MedSAM study runs.

- `tables/`: CSV summaries for metrics, deployment gates, CPU timing, calibration, ablations, and external evaluation.
- `figures/`: PNG visualizations from the study.
- `splits/`: JSON dataset split manifests.

Large generated artifacts are not included:

- `raw/`
- `models/`
- `predictions/`
- `logs/`

Regenerate large artifacts locally from the scripts or notebook when needed.
